# 1.3 웹 표준

필딩은 팀 버너스리를 비롯한 다른 개발자들과 함께 웹 확장성을 높이는 작업을 했다. 이런 작업을 통해 표준 설계안을 만들었고, HTTP의 새로운 버전인 [HTTP/1.1](http://www.rfc-editor.org/rfc/rfc2616.txt) 스펙을 작성하였다. 또한. URI 문법 형식도 [RFC 3986](http://www.rfc-editor.org/rfc/rfc3986.txt)으로 규정하였다. 이렇게 정리된 표준은 웹을 통해 급속히 확상되었고, 웹이 지속적으로 성장하는 기를 마련했다.
