# 2. 균일한 인터페이스

웹을 구성하는 클라이언트, 서버, 네트워크 등의 매체 간 인터페이스는 각 인터페이스의 일관성에 기반한다. 이러한 웹 컴포넌트들 중 하나라도 기존에 구축된 표준에서 벗어나면, 웹 커뮤니케이션 체계는 붕괴된다.

웹 컴포넌트들은 필딩이 구분한 다음 네 가지의 인터페이스 제약에 따라 서로 일관성 있게 상호 운영된다.
