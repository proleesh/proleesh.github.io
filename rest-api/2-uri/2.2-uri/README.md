# 2.2 URI 형태

이 부분에서 언급하는 규칙들은 URI 형식에 적용되는 것들이다. [RFC 3986](http://www.rfc-editor.org/rfc/rfc3986.txt) 에서는 일반적인 URI 문법을 다음과 같이 정의하였다.

> URI = scheme "://" authority "/ " path \["?" query]\["#" fragment]

[**규칙: 슬래시 구분자(/)는 계층 관계를 나타내는 데 사용한다.**](undefined.md)

[**규칙: URI 마지막 문자로 슬래시(/)를 포함하지 않는다**](uri.md)

[**규칙: 하이픈(-)은 URI 가독성을 높이는 데 사용한다.**](uri-..md)

[**규칙: 밑줄(\_)은 URI에 사용하지 않는다**](\_-uri.md)

[**규칙: URI 경로에는 소문자가 적합하다**](uri-1.md)
